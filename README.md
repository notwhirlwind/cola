# 「渴了嗎 來杯可樂吧」可樂資訊圖表專題

## 一、作業簡介

這項專題以「可樂：從藥水到全球產業」為主題，製作一份資訊圖表。
介紹可樂的起源、成分演變、冷知識、經濟影響與文化影響。
並運用Git、GitHub 進行版本管控，記錄資訊圖表從構想到完成的設計過程，包括草圖設計、內容整理、版面調整與最終成品。
---
## 二、版本管控說明

本專案主要追蹤以下內容：
* 資訊圖表設計草稿（sketch）
* 版面配置修改（layout）
* 內容更新（content）
* 最終完成圖（final）

版本變化過程：
1. 初始構想、主題設定
2. 草稿設計
3. 版面設計優化
4. 資料補充與內容修正
5. 視覺美化與最終成果
---
## 三、版本優化與調整過程

在製作過程中，透過多次版本更新進行優化，包括：
